# Diqit Bee KDS multiple platforms

## Technologies 
### Logic
- Electron
- ReactJs 

### UI
- Reactstrap

Start with Electron-React-Boilerplate

## Development
```
yarn dev
```
More info see [Electron-React-Boilerplate](https://github.com/electron-react-boilerplate/electron-react-boilerplate)

## Build

For Windows 
```
yarn package-win
```

For MacOS 
```
yarn package-mac
```

For Linux
```
yarn package-linux
```
